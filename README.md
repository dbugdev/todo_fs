# A Todo App with fs module

## Features

- Clean UI
- No Database Used.


## Installation

Clone it to your pc.
Install the dependencies and devDependencies and start the server.

```
yarn install
yarn dev
```


## Dependencies 

```
"dependencies": {
    "ejs": "^3.1.6",
    "express": "^4.17.1"
  },
  "devDependencies": {
    "nodemon": "^2.0.7"
  }
```
